# Formo

Formo is a minimalistic black-and-white themed React application that empowers users to create, share, and analyze custom forms with ease. Built with a modern tech stack, Formo combines simplicity in design with powerful functionality.

---

## ✨ Features

### 📝 Form Builder
- Create forms with **unlimited fields**.
- Supports multiple field types:
  - Single-line input
  - Multi-line input
  - Dropdown list
  - Checkbox
  - Radio buttons
  - Color palette picker
  - Range slider
- Drag-and-drop **sort-order view** to rearrange form items.

### 🔗 Sharing
- On form submission, automatically generate:
  - A **shareable link**
  - A **QR code**
- Share directly to popular social apps (WhatsApp, Facebook, X/Twitter) — only if installed on the device.

### 📊 Dashboard & Analytics
- View responses in a **clean, well-presented dashboard**.
- Analyze collected data with ease.

### 🔐 Authentication
- Sign in with **Google** or **Email signup with verification**.
- Secure authentication powered by Supabase.

---

## 🛠 Tech Stack
- **React + TypeScript**
- **Vite** (fast build tool)
- **TailwindCSS** (utility-first styling)
- **Supabase** (backend for forms & responses)
